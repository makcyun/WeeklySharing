## 如何选择工具？

关于如何选择工具，不妨参考[科幻小说作家 Bruce Sterling 的清单](http://www.tonyyet.com/tools)：

- 这货是不是漂亮？
- 这货是不是承载着你的情感？
- 这货能否帮助你提升工作效率？

假如它不能满足以上的三个条件，Bruce Sterling 的建议是，毫不犹豫地丟弃它。
